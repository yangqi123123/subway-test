# 武汉地铁保护区管理平台 — 功能清单

| 说明 | 内容 |
|------|------|
| 依据 | 顶栏 `TOP_NAV`、侧栏 `SIDEBAR`、工作台 `WB_MEGA`、全景地图子页跳转 |
| 形态 | 电脑端 Web 为主；入口页含移动端（微信小程序）占位 |
| 层级约定 | **一级**：顶栏模块；**二级**：下拉/侧栏分组/超级卡片；**三级**：具体功能页 |

## 十、补充说明

1. **重复入口**：巡查类、统计类、飞行计划/航线、态势/告警等可从「顶栏」或「我的工作台」进入，业务页面为同一套 HTML。
2. **隐藏/占位**：`am-ops-fulltime`、`am-ops-uav` 等菜单在 Hub 中隐藏但页面保留；移动端为原型占位。
3. **嵌入模式**：项目管理内「巡查」可通过 `?embed=1` 嵌入人工/无人机巡查页，不显示顶栏侧栏。
4. **数据形态**：当前为前端原型，列表多为样例数据，部分导出/筛选为 UI 能力演示。
